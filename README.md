🍔 Swiggy Clone Website

A modern and responsive food delivery web application inspired by Swiggy. Built using HTML, CSS, JavaScript, JSON, and LocalStorage.

✨ Features:

* Responsive UI Design
* Food Categories
* Dynamic Restaurant Rendering
* Search Functionality
* Add To Cart System
* Quantity Update
* Total Price Calculation
* SweetAlert Notifications
* Login & Signup Authentication
* Mobile Friendly Layout

🛠 Technologies Used:
HTML5, CSS3, JavaScript, JSON, LocalStorage
